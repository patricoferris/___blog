Site Info

This website is a MirageOS Unikernel deployed to Google Cloud (using the virtio backend, some Solo5 magic and a Github Action for continuous deployment). The content repo uses a really simple static-site generator that I wrote and pushes the built site to the live branch. The Unikernel then does a git pull using Irmin from this branch to get the latests content. All of this is possible thanks to:

  • @dinosaure's blog for the Irmin parts.
  • @sgrove's site for the Google Cloud deployment idea.
